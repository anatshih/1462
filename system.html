المحور الثاني: مراقبة النظام وسجلاته

1. تحليل السلوك المتقدم (Sysinternals Suite)

هي مجموعة أدوات متقدمة تعمل على نظام Windows تساعد في مراقبة سلوك النظام وتحليل العمليات واكتشاف البرمجيات الخبيثة. لا تخبرك هذه الأدوات مباشرة أن البرنامج خبيث، بل تساعدك على تحليل السلوك لاتخاذ القرار.

  • Procmon: تراقب العمليات وتعديلات السجل (Registry) والملفات.
  • RamMap: تكشف استهلاك الذاكرة العشوائية لكل برنامج.
  • TCPView: تعرض الاتصالات الشبكية الحالية والمنافذ المفتوحة لكل عملية.
  • Autoruns: تكشف البرامج والخدمات التي تعمل تلقائياً عند بدء التشغيل.

2. تحليل سجلات الأحداث (Event Viewer)

يعد Event Viewer بمثابة ذاكرة النظام؛ فهو يسجل كل ما يحدث، ويقسم السجلات إلى أنواع لتسهيل مهمة المحقق:

  • Security (الأمان): يوثق عمليات تسجيل الدخول الناجحة والفاشلة والصلاحيات.
  • Application (التطبيقات): يوثق الأخطاء والانهيارات التي تحدث للبرامج المثبتة.
  • System (النظام): يوثق أحداث نواة الويندوز، تشغيل الخدمات، وتوصيل الأجهزة الطرفية (مثل USB).
